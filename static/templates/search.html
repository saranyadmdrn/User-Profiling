How Crawling?

  • https://docs.oracle.com/javase/tutorial/ and https://en.wikibooks.org/wiki/Java_Programming have been crawled.
  • Scrapy is used for crawling the website. It scrapes the content on the page and stores it into a csv file.
  • Scrapy first scrapes all the links on the main page. Then, it iterates through each link. Under each link, the subsections of the page is scraped separately.
  • The csv file stores the url of the page where the content is present, the topic under which the content appears and the content

How Indexing?

  • Elastic search is used for indexing the crawled content.
  • The csv file generated from crawling is used for indexing.
  • The script will populate the elasticsearch index from the file. Index are created on subsections of the page that has been crawled.
  • bulk() api helpers are used to index these large datasets.
  • bulk() api helper consumes an iterator of actions and sends them to elasticsearch in chunks.
  • When a search is made, elasticsearch is queried to return all the relevant content.
