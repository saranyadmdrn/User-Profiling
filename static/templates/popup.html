Analytics on tags

  • The chart shows the number of tags that the user has interacted with. In addition to this, it also shows the tags collected from the post that other users have interacted with.
  • Number of tags can tell the interest of the user.
  • Tags can be used to help the user identify questions that may be interesting or relevant to the user.
  • The user can see the tags by other users and might identify new topics of interest.
  • The user can also use these analytics to find out the trending topics that the other users are talking about.
  • The user can also identify users of similar interests and can share knowledge with each other.
  • Lesser number of tags may indicate that the particular topic is not trending
